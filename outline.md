* stream ciphers
    - csprngs

    - не более одного сообщения с одним ключом
* block ciphers
    - блоки определённого размера
    - prp
    - более _сильные_
    - ecb и semantic security
    - ~~prf~~
    - det ctr / аналогия с stream cipher
    - cpa / probabilistic algos / randomization
    - cbc / iv / rand ctr
    - malleability + cca
* message integrity
